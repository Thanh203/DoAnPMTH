## Mục đích:
> - Kiếm lợi nhuận

### Mục tiêu:
+ Quản lý sách
+ Quản lý danh mục
+ Quản lý đơn hàng
+ Quản lý khách hàng
+ Đăng nhập/ đăng ký

+ Tìm kiếm nâng cao
+ Đánh giá
+ Thanh toán online
+ Xem trước e-book
+ Lịch sử đọc e-book
+ Giỏ hàng
